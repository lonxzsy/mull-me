# Mull Me

React + TypeScript + Vite приложение

## Технологии

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- React Router

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Линтинг

```bash
npm run lint
```
